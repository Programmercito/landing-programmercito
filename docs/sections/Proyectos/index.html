JwtLinkDinamic

Librería para guardar datos firmados en una URL, utilizando JWT, para ahorrar backends o algún otro tipo de repositorio de backend. Facilita la transmisión segura de datos entre aplicaciones Java y Angular, aprovechando JWT para la autenticidad y seguridad. Disponible en Sonatype Central.
